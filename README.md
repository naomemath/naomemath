# 💛 naomemath · finanças

Dashboard de finanças pessoais com controle de **PF e PJ**, investimentos, transferências e saúde financeira.

## 🚀 Acesso online

Disponível em: `https://naomemath.vercel.app` (após deploy)

## ✨ Funcionalidades

- 👤 Conta **PF** — entradas, contas, investimentos
- 🏢 Conta **PJ** — faturamento, saídas (impostos, contabilidade…)
- 🔄 **Transferências PJ → PF** — pró-labore, distribuição de lucros
- ❤️ **Saúde Financeira** — score e métricas automáticas
- 📊 **Gráficos** — distribuição, evolução, carteira
- 🗂 **Histórico** mensal completo
- 🔒 **Senha de acesso** (padrão: 1234)
- 📦 **Backup/Restore** via JSON

## 📱 Como usar

1. Acesse a URL do Vercel
1. Senha padrão: `1234` (altere em ⚙️ Config)
1. Comece lançando as entradas do mês
1. Exporte backup mensalmente em ⚙️ Config

## 🗄️ Dados

Os dados ficam no **localStorage** de cada dispositivo.
Para sincronizar celular ↔ computador: **Exportar JSON** em um → **Importar JSON** no outro.
